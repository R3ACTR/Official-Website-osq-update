# **App Name**: NeoFolio

## Core Features:

- Hero Section: Hero section with name and profession in large, bold typography.
- Projects Display: Grid layout to display the works with title, description, and link. Keep the design clean and clear with sharp edges and strong borders.
- About Section: Brief about section presenting relevant information.
- Contact Form: Straightforward contact form.
- Theme matching for suggestions: AI powered tool that analyzes the text of the about section and identifies the main themes. Generates suggestions of other projects from the Project Display whose description match the themes.
- Responsiveness: Responsive layout adapting to different screen sizes.

## Style Guidelines:

- Background color: Light gray (#E0E0E0) to provide a subtle contrast.
- Primary color: Deep blue (#3F51B5) to stand out.
- Accent color: Vibrant orange (#FF5722) to highlight interactive elements.
- Body text: 'Inter', a grotesque-style sans-serif for body text.
- Headline font: 'Space Grotesk', a proportional sans-serif font. Its computerized, techy, scientific feel gives a brutalist edge. Use 'Inter' for body.
- Use minimal icons with a raw, outlined style for essential navigation elements.
- Grid-based layout with visible borders and sharp edges.
- Abrupt transitions, no easing.
